# Cinema-an

Final project for class. A movie recommender that mixes NLP and collaborative filtering.

## How to run

1. Make sure both files in this folder are in the same directory
2. Open Cinema-an.ipynb in Jupyter
3. Hit Run All

No dataset downloads needed. The trained model state is saved in cinema_artifacts.pkl.gz and the notebook loads and decompresses it directly.

## Requirements

Standard Python with these packages (all included in Anaconda):

- numpy
- pandas
- matplotlib
- scikit-learn

## Files

- Cinema-an.ipynb: the notebook
- cinema_artifacts.pkl.gz: pre-trained model state, gzipped to fit GitHubs 25 MB upload limit
